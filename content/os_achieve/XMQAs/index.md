**Name of the Dataset:** XMQAs: A complex-modified question-answering dataset

**Dataset Introduction:** XMQAs is a 1000K complex-modified question-answering dataset based on SimpleQuestions and WebQuestions. Each original simple-expressed question is expanded into a complex-modified question with one or more modifiers applied to different parts of the question. 

**Download Link:** https://drive.google.com/drive/folders/14R7QB6ScmgfSJREMFhWCfv1kd51ofFYm?usp=drive_link

**Relevant Paper:** XMQAs: Constructing Complex-Modified Question-Answering Dataset for Robust Question Understanding
